# تقرير المشروع — Bid OS (منصّة تحليل المناقصات)

## 1) أدوات صناعة الموقع
- **البنية:** Monorepo بـ Turborepo + pnpm.
- **الواجهة:** Next.js 14 (App Router) + TypeScript + Tailwind CSS + Framer Motion + lucide-react.
- **الخادم:** Next.js Server Actions + Route Handlers.
- **قاعدة البيانات:** PostgreSQL (Supabase) + Prisma ORM.
- **الذكاء الاصطناعي:** Anthropic Claude SDK (تحليل الكراسات بإدخال PDF مباشر).
- **التخزين:** S3/R2 (AWS SDK + روابط موقّعة).
- **المستندات:** unpdf (قراءة PDF) · docx (توليد Word) · qrcode (QR للفاتورة) · sharp (الشعار).
- **التحقق:** Zod · **الاختبارات:** Vitest · **النشر:** Vercel + GitHub.

## 4) ماذا استُخدم للحماية
- **عزل المستأجرين:** كل استعلام مقيّد بـ `tenant_id` (مستحيل تسرّب بيانات بين الشركات).
- **الصلاحيات (RBAC):** 4 أدوار مفروضة على الخادم + بوابة أدمن المنصّة.
- **تخزين خاص:** لا ملف عام؛ التنزيل بروابط موقّعة مؤقتة محمية بالعزل.
- **التشفير:** كلمات المرور (bcrypt) · رموز OTP (sha256) · رموز الجلسات (مجزّأة).
- **الكوكيز:** httpOnly + Secure + SameSite.
- **التحقق من المدخلات:** Zod على كل عملية.
- **ترويسات أمان:** X-Frame-Options, nosniff, Referrer-Policy, Permissions-Policy, HSTS، وإخفاء X-Powered-By.
- **توثيق جوال (OTP)** لمكافحة الإساءة + **سجل تدقيق (AuditLog)** للعمليات الحساسة.
- **الأسرار في متغيّرات بيئة فقط** (لا أسرار داخل الكود).

## 5) كم كلّف من اشتراكات وغيره
**تكلفة البناء = 0 ريال** (كل الخدمات على الطبقات المجانية):
| الخدمة | التكلفة |

| GitHub · Supabase · Cloudflare R2 · Resend | مجانية |

## 7) هل ملف env خاص ومحمي؟
- ✅ ملف `.env` **غير موجود في GitHub** (مستبعَد عبر `.gitignore`)، **ولا على القرص حاليًا**.
- ✅ أسرار الإنتاج محفوظة **مشفّرة داخل Vercel** (موسومة Sensitive) — **لا أستطيع قراءتها، ولم أرَ أي قيمة منها**.
- ✅ المرفوع الوحيد هو `.env.example` — قالب فارغ بلا أي سرّ.
- ⚠️ **بصراحة:** لو أنشأت `.env` محليًا على جهازك، فهو ملف نصّي عادي على قرصك — أي أداة لها وصول لجهازك (بما فيها أدواتي **لو فتحتُه بطلبك**) تقدر تقرأه. حمايته تعتمد على أمان جهازك لا على Git. لكن أسرار الإنتاج في Vercel مشفّرة وخارج متناولي تمامًا.
